Thank you for using Marathon.

We hope you will find Marathon useful in your projects. Marathon is one of the most widely used test automation solution for Java/Swing, Java/FX applications.

Marathon support is provided through Marathon Java GUI Testing group.

Say hello to us there... please drop a message if you face any issues.

Some of the following links describe MarathonITE, a commercial version that is developed and supported by Jalian Systems. MarathonITE provides more features including editors for objectmap, objectmap configuration, features, suites and also various views of features, suites, issues etc. to group test scripts. Parts of the information should be useful for Marathon also.
